Simple URL Shortener API Microservice

How to use the service:

  • Point your browser or your app to the following path on our domain: /_api/urls/. After this path you should put a valid URL you want to be shortened. For example, http://www.google.com.
  • The API will respond with a JSON object with two fields - the shortened URL and the original URL. For example: 
            		    			
{
    "shortUrl":"https://url-shortener-stellarnode.c9users.io/B",
    "originalUrl":"http://www.google.com"
}
  • Alternatively you can use the form below to enter a URL you want to shorten. It will be appended to the path mentioned above.
  • Once you push the button, we will show you the output of your query in the code block below. The API responds with a JSON string. If the URL format in the request is not valid or is not recognized, a JSON object with Invalid URL value is returned.
